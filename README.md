# Проектирование высоконагруженного сервиса стимингового кинотеатра - Кинопоиск
## Часть 1. Тема, функционал и аудитория

### Тема
Тема данной курсовой работы - проектирование высоконагруженного сервиса стимингового кинотеатра на примере [Кинопоиска](https://www.kinopoisk.ru/)
### Целевая аудитория
- Общее число пользователей - 60 млн. [^1]
    - из них 18 млн - смотрящих по мультиподписке Яндекс Плюс в месяц  [^1]
    - в среднем каждый день сайт посещают 5 млн пользователей [^2]
- 75% пользователей являются гражданами РФ, дальше по количеству пользователей идут Беларусь, Казахстан и Германия  [^2]
- самая многочисленная аудитория - 25-34 года - 25% от общего числа пользователей. [^2]

### Ключевой функционал 
- Поиск и просмотр фильмов и сериалов
- Доступ в кинотеатру по мультиподписке Яндекс Плюс
- Регистрация и авторизация пользователей
- Возможность оставить оценку, отзыв, рецензию на фильм, сериал.
